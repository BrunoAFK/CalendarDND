# New
- Added a new meeting mode: **Vibrate**.
- You can now choose between **Vibrate**, **Priority Only**, and **Total Silence** for what should happen during meetings.

# Improvements
- Vibrate mode now switches the phone to vibrate for the active meeting window and restores your previous ringer mode afterward.
- The app now respects manual ringer changes during a meeting, just like it already respects manual DND changes.
- Vibrate mode no longer depends on DND policy access, so it can work even if full Do Not Disturb control is not granted.
- Updated the DND Mode screen and related text to better reflect the new meeting mode options.

# Security
- Improved dependency security by updating and overriding vulnerable build and app libraries.
